### 소수란

소수란 2보다 큰 자연수 중에서 1과 자기 자신을 제외한 자연수로는 나누어떨어지지 않는 자연수 이다.

이런 소수를 판별하는 알고리즘으로는 다음과 같다.

------------------------------------------------

### CODE

```C++
bool isPrimeNumber(int x) {
    for (int i = 2; i <= x; i++) {
        if (x % i == 0) {
            return false;
        }
    }
    return true; 
}
```

-----------------------------------------------------

### 설명1

위 코드의 시간 복잡도는 O(n)이다. 예를 들어 1000000이라는 수가 소수 인지 확인해야 할 떄는 1000000을 2부터 모든 수에 대하여 하나씩 나누어야 한다.

따라서 이와 같이 알고리즘을 작성하면 비효율적이다.

---------------------------------------------------------

### 더 효율적으로

16이라는 수의 약수는 1,2,4,8,16이다.

여기서 수학적으로 봤을때 간단하게 판단할 수 있는 것은 가운데 제곱근 숫자를 기점으로 약수가 대칭적이라는 것을 알 수 있다.

1 * 16, 2 * 8, 4 * 4, 8 * 2, 16 * 1 처럼 말이다.

즉 효율적으로 제곱근(가운데) 숫자까지만 비교하면 되므로 개선한 코드는 다음과 같다.

-------------------------------------------------

### CODE2

```C++
bool isPrimeNumber(int x) {
    for (int i = 2; i <= (int) sqrt(x); i++) {
        if (x % i == 0) {
            return false; 
        }
    }
    return true;
}
```

-----------------------------------------------------

### 설명2

위와 같이 코드를 작성하면 설명1에서와 같은 n의 입력값 1000000이 들어오면 1000까지만 확인하면 되는 것이다.

시간 복잡도는 O(N^1/2)인 것을 알 수 있다.
