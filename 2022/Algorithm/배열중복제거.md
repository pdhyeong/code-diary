### 간단한 알고리즘

중복이 있는 배열에서 중복을 제거 하고 오름차순으로 정렬하는 알고리즘

```C
include<stdio.h>
int main(){
  int arr[10] = { 10,9,9,8,7,7,7, 6,5, 1 };
	int b[10]; // 중복을 제거한 데이터를 담을 배열
	int i, n = 1;
	b[0] = a[0]; // 첫 데이터를 a[0]의 데이터를 삽입

 // 중복이 되지 않는 값만 a[i]에서 걸러서 b[n]에 삽입하고 [n++]연산으로 한칸 전진한다.
	for (i = 1; i < 10; i++)
		if (arr[i] != b[n - 1]) b[n++] = arr[i];
    
  // 버블 정렬
  for(i = 0 ;i<n;i++)
  {
    for(int j =0;j<n-i;j++)
    {
      if(b[j] > b[j+1]) {
        temp = b[j];
        b[j] = b[j+1];
        b[j+1] = temp;
      }
    }
  }
  
  // 배열출력
	for (i = 0; i < n; i++)
		printf("%d ", b[i]);
  return 0;
}

```
